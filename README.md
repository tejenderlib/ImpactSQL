# 🌍 Global Disaster Impact Analysis

This project is currently **in progress**.

I’m building a data analysis pipeline using **MySQL** and will later create a dashboard in Looker Studio.

Right now, the focus is on:

* importing raw data
* preparing a reliable dataset for analysis

---

## 🔄 Workflow

```text
Raw CSV Data 
↓ 
Import into MySQL (disasters_raw) 
↓ 
Data Cleaning & Transformation 
↓ 
Final Structured Table (disasters) 
↓ 
Connect to Looker Studio 
↓ 
Dashboard Creation & Insights
```

* `disasters_raw` → stores raw data (no changes)
* `disasters` → cleaned and structured table
